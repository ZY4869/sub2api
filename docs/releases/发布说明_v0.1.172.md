# v0.1.172 更新说明

发布日期：2026-03-31

## 重点更新

- 新增 Google Batch Archive 全局设置入口，支持配置轮询节奏、40 小时预下载窗口、下载超时、清理频率和 VPS 本地归档根目录。
- Google 账号创建与编辑页新增 Batch Archive / Overflow 配置项，可分别控制 AI Studio 归档、自动预下载、保留天数、本地下载计费模式，以及 AI Studio 溢出到 Vertex、Vertex 接收溢出的行为。
- 使用量日志新增 Batch 相关操作类型与扣费来源字段，后台与用户侧用量页可直接区分 `batch_create`、`batch_settlement`、`local_archive_download` 等操作，并显示是否扣费。

## 后端能力补充

- 接入 Google Batch Archive 相关设置视图、管理接口、仓储与运行时服务，扩展 Batch 路由与 AI Studio v1beta Batch 处理链路，为后续本地归档、轮询、预下载和透明回源提供基础支撑。
- 扩展 usage log / billing 写入链路，支持记录 `operation_type`、`charge_source` 等批量任务审计信息。
- 修复 `wire_gen_test` 注入参数基线问题，恢复后端整仓测试可通过。

## 前端联动

- 后台设置页新增 Google Batch Archive 设置卡片。
- 账号创建/编辑弹窗新增 Google Batch Archive 专用编辑器，并完成创建、重置、回填与提交逻辑接线。
- 管理员与用户用量页统一批量操作展示逻辑，补充对应中英文文案。
