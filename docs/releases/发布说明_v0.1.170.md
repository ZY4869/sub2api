# v0.1.170 更新说明

发布日期：2026-03-31

## 重点修复

- 修复 OpenAI compat model 规范化与 `gpt-5.4-xhigh` 路由问题，确保消息转发、映射模型和 usage/billing 记录继续按用户请求模型计费。
- 修复 OAuth 生命周期稳定性问题：`TokenRefreshService.Stop()` 幂等，缺失 `refresh_token` 时不再误触发临时停调度，管理员重置账号状态时会同步清理 temp unsched。
- 修复 Claude 相关协议细节，包括 `X-Claude-Code-Session-Id` 头重写、模型定价远端地址更新，以及软删除 API Key 后可重新复用同一 key。

## 功能增强

- Anthropic OAuth / SetupToken 账号新增 `user:file_upload` scope。
- 管理端账号创建/编辑支持为 Anthropic OAuth / SetupToken 账号配置自定义转发 URL，并继续保留本地拆分后的表单与服务结构。
- 用户端与管理员端 Token Usage Trend 图表新增 cache hit rate 展示。
- 修复 Safari 下 Ops System Log 筛选下拉框显示异常。
