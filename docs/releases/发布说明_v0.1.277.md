# v0.1.277 更新说明

发布日期：2026-04-23

## 修复

- 修复 OpenAI `compat` 图片桥接在 ChatGPT OAuth/Codex 上游下可能触发 `Instructions are required` 的问题：对 `/v1/images/*` compat 桥接到 `/v1/responses` 的上游请求补齐 Codex OAuth transform（默认 `instructions`、强制 `stream/store` 等）。
- 修复 `/v1/images/*` compat 在上游返回错误时可能出现 **重复写入响应**（拼接两段 JSON）的问题。
- 修复 SSE 探测误判：避免把包含 `data:image/...` 的 JSON 响应误判为 SSE，导致解析失败。
- 修复 `/v1/responses model=gpt-image-2` 简写在上游返回版本化 model（如 `gpt-5.4-mini-2026-03-17`）时未能替换回 `gpt-image-2` 的问题。
