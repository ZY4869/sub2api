# v0.1.278 更新说明

发布日期：2026-04-24

## 修复 / 优化

- 修复 OpenAI Pro 的 Spark 与普通 Codex 额度归因混用：两侧额度独立统计、独立冷却，避免互相污染。
- 后台账号列表用量 UI 升级：用量窗口列移除“剩余…重置于…”，重置信息统一在“重置日期”列以紧凑形式展示（倒计时 + 绝对时间）。
- 账号调度优先级修复：在不破坏粘性会话的前提下，新增全局“并发占用率（Current/Max）更低优先”策略，缓解单账号并发过大导致的拥塞。
- 修复百度智能文档（Baidu Document AI）legacy 平台值兼容：`platform=baidu` 时也展示官方令牌输入区域。
- gpt-image-2 / `image_generation` 兼容增强：补齐 `quality/background/output_format/output_compression/n` 等参数，且对 `n/output_compression/partial_images` 支持字符串数字归一并在网关侧拦截非法值（避免 `tools[0].n` 类型错误）。
- 上游非流式响应读取上限默认提升到 `1024MB`（可通过 `gateway.upstream_response_read_max_bytes` 覆盖）；请求记录补齐并展示生成图片尺寸（`media_resolution`）。
